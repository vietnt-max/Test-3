# Webtile-like Next.js Project

Scaffolded Next.js project that mimics a simple template grid (like chonweb.vn).

## Quick start

Install deps:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Deploy

1. Create a GitHub repo and push this project.
2. Go to vercel.com, import the GitHub repository and deploy (Vercel detects Next.js automatically).

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel login
vercel
```
